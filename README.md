## Obsidian: It's MyVault!!!!!

这是一个用于管理和同步我的笔记的存储库。

### 目录结构

- `Dataview-Example/`: 包含使用 Dataview 插件的示例笔记。
  - `Note1.md`
  - `Note2.md`
  - `Playground.md`
- `Excalidraw/`: 用于存储 Excalidraw 图形文件。
- `README.md`

### 插件

- Dataview
- Excalidraw
- Git
- Linter
- Style Settings
- Better Export PDF

### 安装和设置

1. **克隆存储库**

```bash
git clone https://github.com/shinoharaharuna/MyVault
```

2. **进入目录**

```bash
cd MyVault
```

3. **在 Obsidian 中打开**

- 打开 Obsidian 应用。
- 选择“打开文件夹为库”，然后选择克隆的目录。

### 注意事项

- 确保 `.gitignore` 文件中包含 Obsidian 的临时文件和其他不需要同步的内容。
- 定期备份 Vault，以防数据丢失。
